Sobre mim

Olá, meu nome é Paulo e trabalho diretamente com tecnologia da informação desde 2009, depois de desistir de uma carreira como funcionário público para seguir essa minha vocação. Bacharel em Administração de Empresas por formação, sou também pós-graduando em Engenharia de Software. Já atuei em diversas posições diferentes, mas atualmente trabalho com análise de requisitos em projetos Web e Mobile desde 2013. Sou um entusiasta de tecnologia desde criança, sempre procurando aprender mais e mais. Eu gosto de diversas tecnologias, desde programação à hardware, redes, sistemas operacionais e sou apaixonado por livros

Portfólio

A maioria dos projetos em que eu trabalhei não são públicos devido ao fato de eu prestar serviços a companhias. Porém posso relatar que foram projetos relacionados à Gestão de Compras, Gestão de Tickets de TI, Geração e Manutenção de Contratos, Aplicativos móveis, APIs, entre outros. Este site é um dos meus permantentes projetos. Ele utiliza tecnologia Bootstrap para o layout responsivo, Gulp para minificação de arquivos e melhoria de performance entre outras e é constantemente atualizado. Implanta também outras tecnologias como Google Analytics, Google Translate, etc. O próprio código fonte deste site está disponível no meu perfil no Github.

Os projetos listados abaixo foram desenvolvidos de forma independente como freelancer. Sejam para clientes ou como forma de evoluir no aprendizado da programação.

TelerespondeAplicativo móvel

Teleresponde é uma pequena empresa da minha cidade natal, Piracicaba, que agrega informação de parceiros comerciais da cidade e região, com alto grau dfe confiabilidade. Provendo informações como números de contato e endereço a centenas de usuários.

com versões para Android e iOS, o aplicativo usa tecnologia Cordova e demais tecnologias Web como HTML, CSS e Javascript para buscar informação em uma API Rest e exibir resultados, utilizando também a API do Google Maps.
